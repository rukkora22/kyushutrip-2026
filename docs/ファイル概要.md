## script.js

### 機能一覧

| 機能 | 概要 |
|------|------|
| ヘッダー切り替え | Heroセクション通過後に白背景に切り替え |
| ドロワーメニュー | 開閉・オーバーレイタップで閉じる |
| Day タブ | Day1〜3の切り替え |
| タイムラインリンク | クリックで該当スポットカードへスムーズスクロール |
| モーダル | `SPOTS` オブジェクトのデータから動的に生成 |
| モーダル閉じる | ×ボタン・画面外タップ・Escapeキー |
| スクロールリビール | IntersectionObserver でフェードイン（スタガー対応） |
| 画像フォールバック | 読み込み失敗時に非表示（エラー表示なし） |
| Instagram | 埋め込み失敗・URL未設定時は何も表示しない |

### SPOTS オブジェクト（script.js 内）
スポットデータは `script.js` 冒頭の `SPOTS` 定数に集約されています。
Instagram URLや「やることリスト」を追加する場合は該当スポットの `igUrl` / `todo` を編集してください。

```js
// 例：松むらのInstagram URLを設定する場合
matsumura: {
  igUrl: 'https://www.instagram.com/matsu3981/',
  todo:  ['馬刺しを食べる', 'ランチを楽しむ'],
  // ...
}
```
